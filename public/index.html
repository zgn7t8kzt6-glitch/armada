"Data coverage" shows, field by field, how many clients have each piece pulled from Kipu and where it comes from. "Sync roster" adds/updates the live active census. "Rebuild" wipes the current client list and reloads it fresh from Kipu — use it to clear stale imports.

Salesforce

        
Connect Salesforce (system of record for inbound referrals) via a Connected App using the OAuth client-credentials flow. Paste the three values below and Test.

        

          

          

        

        

          

          

        

        

          
          
          
          
          
          
        

        

        
In Salesforce: Setup → App Manager → New Connected App → enable OAuth, add scopes api + refresh_token, check Enable Client Credentials Flow, save. Then Manage → Edit Policies → set a “Run As” integration user. Get the key/secret under Manage Consumer Details. Instance URL is your My Domain (Setup → My Domain). After connecting, click Discover objects so we can map your referral data.
