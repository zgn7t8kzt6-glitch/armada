# Authoritative Regulatory Source Register

Blueprint §36: implementation must be validated against current authoritative
sources at implementation time. This register tracks them. Store citations,
retrieval dates, and mappings — never unlicensed copies of copyrighted
standards (e.g. Joint Commission manuals).

This register is an engineering control. It is not legal advice and does not
by itself establish compliance (CLAUDE.md #16).
